Question:
- If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?

Answer:
- JAVA không cho phép điều này.

Giải thích:
- Java có sử dụng cơ chế nạp chồng phương thức (overloading constructor).
- Overloading cho phép một lớp có khả năng định nghĩa ra nhiều phương thức có cùng tên, nhưng khác nhau về tham số truyền vào.
- Nhưng tuy nhiên ở đây, 2 constructor trong ví dụ trên lại trùng chữ ký nên JAVA sẽ không cho phép điều đó.
